# Lập lịch tác vụ tự động với Crontab

## I. Cron là gì?

* **Cron:** Dịch vụ chạy ngầm (`crond`) — theo dõi thời gian và tự động thực thi lệnh đã lên lịch.
* **Crontab:** File văn bản lưu danh sách các lịch trình. Mỗi user có 1 file crontab riêng.

---

## II. Cấu trúc một dòng Crontab

```
*     *     *     *     *     /đường/dẫn/lệnh
│     │     │     │     │
│     │     │     │     └── Ngày trong tuần (0-6, CN=0)
│     │     │     └──────── Tháng (1-12)
│     │     └────────────── Ngày trong tháng (1-31)
│     └──────────────────── Giờ (0-23)
└────────────────────────── Phút (0-59)
```

### Ký tự đặc biệt

| Ký tự | Ý nghĩa | Ví dụ |
|:---|:---|:---|
| `*` | Mọi giá trị | `* * * * *` = mỗi phút |
| `,` | Danh sách | `1,15,30` = phút 1, 15, 30 |
| `-` | Khoảng | `1-5` = thứ Hai đến thứ Sáu |
| `/` | Chu kỳ | `*/5` = mỗi 5 phút |

### Macro rút gọn

| Macro | Tương đương | Ý nghĩa |
|:---|:---|:---|
| `@reboot` | — | Chạy 1 lần sau khi khởi động |
| `@hourly` | `0 * * * *` | Mỗi giờ |
| `@daily` | `0 0 * * *` | Mỗi ngày lúc 00:00 |
| `@weekly` | `0 0 * * 0` | Mỗi Chủ nhật lúc 00:00 |
| `@monthly` | `0 0 1 * *` | Ngày đầu mỗi tháng lúc 00:00 |

## III. Lệnh quản lý Crontab

```bash
crontab -e           # Tạo hoặc chỉnh sửa crontab
crontab -l           # Xem danh sách crontab hiện tại
crontab -r           #  Xóa toàn bộ crontab (không hỏi xác nhận)

# Quản lý crontab của user khác (cần quyền admin)
sudo crontab -u dev1 -l    # Xem crontab của dev1
sudo crontab -u dev1 -e    # Sửa crontab của dev1
```

## IV. Ví dụ thực tế

```plaintext
# 1. Backup thư mục mỗi ngày lúc 2:30 sáng
30 2 * * * tar -zcf /backup/project-$(date +\%F).tar.gz /opt/project_alpha

# 2. Kiểm tra trạng thái Nginx mỗi 5 phút
*/5 * * * * systemctl is-active nginx >> /var/log/nginx_health.log

# 3. Dọn file tạm vào 12h trưa thứ Hai đến thứ Sáu
0 12 * * 1-5 /home/chien2003/scripts/clean_temp.sh

# 4. Thông báo khi server reboot
@reboot echo "Server đã reboot" | mail -s "System Alert" admin@example.com
```

## V. Lưu ý quan trọng

**1. Luôn dùng đường dẫn tuyệt đối:**
```plaintext
 python3 script.py
 /usr/bin/python3 /home/chien2003/script.py
```

**2. Redirect output để lưu log lỗi:**
```plaintext
* * * * * /home/chien2003/script.sh >> /var/log/cron.log 2>&1
```

**3. Kiểm tra log cron daemon:**
```bash
grep -i cron /var/log/syslog | tail -20   # Ubuntu
journalctl -u cron.service -n 20          # Journald
```
