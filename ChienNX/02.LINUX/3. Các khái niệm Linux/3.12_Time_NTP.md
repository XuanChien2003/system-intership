# Cấu hình Thời gian và NTP Server

## I. Tại sao phải cấu hình thời gian đúng?

Thời gian hệ thống ảnh hưởng trực tiếp đến: **Log hệ thống**, **xác thực bảo mật (SSH/SSL)**, **crontab** và **đồng bộ dịch vụ mạng**.

---

## II. Quản lý thời gian với `timedatectl`

```bash
# Xem trạng thái thời gian hiện tại
timedatectl

# Xem danh sách múi giờ
timedatectl list-timezones
timedatectl list-timezones | grep -i Asia   # Lọc châu Á

# Đặt múi giờ Việt Nam
sudo timedatectl set-timezone Asia/Ho_Chi_Minh

# Bật/tắt tự động đồng bộ NTP
sudo timedatectl set-ntp true
sudo timedatectl set-ntp false

# Đặt thời gian thủ công (phải tắt NTP trước)
sudo timedatectl set-time "2026-06-29 09:00:00"
```

## III. Cấu hình NTP Client trên Ubuntu (`systemd-timesyncd`)

Ubuntu tích hợp sẵn `systemd-timesyncd` — nhẹ và đơn giản.

```bash
# Bật đồng bộ NTP tự động
sudo timedatectl set-ntp true
```

Chỉnh file cấu hình để dùng NTP Server Việt Nam:
```bash
sudo vim /etc/systemd/timesyncd.conf
```

```ini
[Time]
NTP=0.vn.pool.ntp.org 1.vn.pool.ntp.org time.google.com
FallbackNTP=ntp.ubuntu.com
```

```bash
# Áp dụng cấu hình
sudo systemctl restart systemd-timesyncd

# Kiểm tra trạng thái đồng bộ
systemctl status systemd-timesyncd
timedatectl show-timesync          # Xem chi tiết NTP server và độ lệch thời gian
```

## IV. Cấu hình NTP Client trên Rocky/CentOS (`chrony`)

Rocky Linux/CentOS dùng `chrony` — chính xác cao, hỗ trợ cả client và server.

```bash
# Cài chrony (nếu chưa có)
sudo dnf install chrony -y
```

Thêm NTP Server Việt Nam vào file cấu hình:
```bash
sudo vim /etc/chrony.conf
```

```plaintext
pool 0.vn.pool.ntp.org iburst
pool 1.vn.pool.ntp.org iburst
# iburst = đồng bộ nhanh ngay khi dịch vụ khởi động
```

```bash
# Bật và khởi động dịch vụ
sudo systemctl enable chronyd
sudo systemctl start chronyd
sudo systemctl restart chronyd   # Sau khi sửa file .conf

# Kiểm tra các NTP server đang kết nối
chronyc sources -v
# Dấu ^* = server chính đang đồng bộ

# Kiểm tra độ lệch thời gian chi tiết
chronyc tracking
```
